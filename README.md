# dk_exchange
2012/03 非正式專案，大一時製作，主發性的製作學生和老師間的互動平台，功能包括註冊、留言、修改、刪除、搜尋、上傳檔案、寄信等。 無純前端頁面。
<ul>
<li>無刷新(Refresh)式留言為特色</li>
<li>防範sql injection攻擊</li>
<li>原生Html、Css、JQuery、PHP製成</li>
<li>無遵守程式撰寫規範(當時還不知道)</li>
</ul>
